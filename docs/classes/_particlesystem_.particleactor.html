checkOverlap

  • checkOverlap(targetClass: any, handler?: Function): boolean

  • Inherited from Actor.checkOverlap

    • Defined in Actor.ts:166

    Collision handling method.

    Returns true if an overlap is found. Uses the .drawing property on the actor to check for overlaps between quads. Call during the Actors update function.

    update()
{
     this.checkOverlap(Enemy, (e : Enemy)=>{
          console.log("overlapping enemy!");
          e.kill();
     });
}

    Parameters

    • targetClass: any

      The class of actors to check this actor against.

    • Optional handler: Function

      callback function if an overlap is found.

    Returns boolean
